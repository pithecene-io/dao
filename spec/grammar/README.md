# Grammar

This directory contains the parser-facing reference grammar for Dao.

Files here are technical reference material rather than constitutional policy.
The normative syntax guarantees live in `docs/contracts/CONTRACT_SYNTAX_SURFACE.md`.

Contents:
- `dao.ebnf` — early reference grammar for the Dao surface language
- `dao.lex` — lexical surface and reserved token inventory
- `indentation_rules.md` — offside-rule notes and arrow taxonomy
